# ML for LLM Mastery 

A structured, from-scratch curriculum covering everything you need to understand and build Large Language Models — from linear algebra to transformer architectures.

> **Philosophy:** Every LLM concept has a simpler origin. Attention = weighted average. Transformer = stacked attention blocks. Pre-training = next token prediction. Understanding the basics deeply makes the advanced trivial.

---

##  Curriculum Structure

Each module has:
- `notes.md` — Theory, intuition, key formulas
- `code.py` — Practical Python implementation from scratch
- `exercise.py` — Hands-on exercise (with solution)
